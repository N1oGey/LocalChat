# How install Echat? 

Install [Termux](https://play.google.com/store/apps/details?id=com.termux) and use this commands

```bash
pkg update
termux-setup-storage
pkg install termux-api
pkg install python
pkg install git
git clone https://github.com/N1oGey/LocalChat.git
cd LocalChat
python LocalChat.py
```
